# projectThree

This project requires [maven](https://maven.apache.org/index.html) to run. For help with installing maven, please see the [maven install guide](https://maven.apache.org/install.html).

### Build Project
```
$ mvn clean install
```

### Start Application
```
$ mvn spring-boot:run
```
